This repository contains the client-side frontend of a media platform built with Vue.js.
It consumes API data from a Laravel backend and displays media content for users.

## Features
- Vue.js component-based UI
- API-driven data rendering
- Media listing and viewing
- SPA-style frontend structure

## Tech Stack
- Vue.js
- JavaScript
- HTML
- CSS

## Project Setup
npm install

### Development
npm run serve

### Production Build
npm run build

## Notes
This is a learning project focused on frontend structure and API integration.
The backend is handled in a separate Laravel repository.
